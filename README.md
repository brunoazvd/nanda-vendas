## nanda-vendas

Este projeto é uma aplicação simples de controle de vendas, criada para facilitar o registro e o acompanhamento de pedidos realizados, entregues e pagos para minha namorada.

Stack utilizada:

- Frontend: React + Material UI + Vite
- Backend: Node.js (Express) + Prisma ORM
- Banco de dados: MySQL

O objetivo é oferecer uma interface prática para cadastro e atualização de pedidos, organizando-os entre "abertos" e "finalizados".
